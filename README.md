# Ejercicios de Repaso - Parcial 1 (Java)

Colección de **20 ejercicios en Java** diseñados para reforzar conceptos fundamentales de programación como:

* Condicionales
* Ciclos
* Operaciones numéricas
* Validación de datos
* Modularización del código

El programa está organizado en **5 partes**, cada una agrupando ejercicios relacionados con un tema específico.

# Parte 1 - Ciclos y Operaciones Numéricas

**Ejercicio 1**
Programa que lee un número `n` y muestra la **suma de todos los números divisibles entre 4** desde `1` hasta `n`.

**Ejercicio 2**
Programa que calcula el **factorial de un número ingresado por el usuario**.

**Ejercicio 3**
Programa que solicita una cantidad `n` de números y muestra cuántos son:

* Positivos
* Negativos
* Ceros

**Ejercicio 4**
Programa que lee un número `n` y calcula el **promedio de los números pares entre 1 y n**.

**Ejercicio 5**
Programa que calcula la **suma de los dígitos de un número** ingresado.

---

# Parte 2 - Condiciones y Divisibilidad

**Ejercicio 6**
Programa que determina si un número es **múltiplo de 7 o de 11**.

**Ejercicio 7**
Programa que muestra los números entre `1` y `n` que son:

* Divisibles entre **3**
* Pero **no divisibles entre 2**

**Ejercicio 8**
Programa que recibe una **hora en formato de 24 horas** y determina si corresponde a:

* Mañana
* Tarde
* Noche

---

# Parte 3 - Uso de Switch

**Ejercicio 9**
Programa que convierte un número del **1 al 5** a su equivalente en **números romanos**.

**Ejercicio 10**
Programa que permite seleccionar una operación mediante una opción:

1. Sumar
2. Restar
3. Multiplicar

Luego realiza la operación con **dos números ingresados por el usuario**.

---

# Parte 4 - Análisis Numérico

**Ejercicio 11**
Programa que lee **dos números** y muestra:

* El mayor
* El menor
* La diferencia absoluta entre ambos

**Ejercicio 12**
Programa que muestra la **tabla de división de un número** desde `1` hasta `10`.

**Ejercicio 13**
Programa que cuenta la **cantidad de dígitos pares** que tiene un número.

**Ejercicio 14**
Programa que determina si un número es **triangular**.

**Ejercicio 15**
Programa que solicita `n` números y calcula el **promedio de los números negativos**, si existen.

---

# Parte 5 - Patrones y Transformaciones

**Ejercicio 16**
Programa que muestra los **primeros `n` múltiplos de 6**.

**Ejercicio 17**
Programa que determina si un número es **potencia de 2**.

**Ejercicio 18**
Programa que **invierte el orden de los dígitos** de un número.

**Ejercicio 19**
Programa que determina si la **suma de los números desde 1 hasta n** es **par o impar**.

**Ejercicio 20**
Programa que convierte una **temperatura de Celsius a Fahrenheit**.

# Extras Implementados

Además de los ejercicios solicitados, se añadieron mejoras para mejorar la estructura del programa:

### Menú de Partes

Se implementó un **menú principal** que permite seleccionar cada una de las partes del programa, facilitando la navegación entre los ejercicios.

### Modularización por Métodos

Cada parte del programa está organizada en su **propio método**, lo que mejora la estructura y facilita la lectura del código.

### Método Utilitario

Se implementó un **método utilitario para la lectura de números**, encargado de:

* Validar entradas del usuario
* Evitar errores cuando se ingresan datos inválidos
* Reutilizar código en diferentes ejercicios

Esto permite que el programa tenga un diseño **más limpio, organizado y modular**.
